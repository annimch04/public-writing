# Archive

This directory holds public source artifacts that support the provenance of Anni McHenry's writing.

Archive material is not the same thing as canon. Essays, edited pieces, and public reading surfaces remain the canonical expression of developed ideas. Archive material preserves lineage: where an idea appeared, how it moved, what it touched, and when it became part of a larger authored body of work.

Examples of archive material may include:

- public social posts
- thread fragments
- exported public records
- source indexes
- migration notes
- provenance maps

The archive should remain separated from the main reading categories so comprehensive records do not overwhelm curated work.
