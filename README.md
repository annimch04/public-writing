# Public Writing

This repository is the canonical source archive for public writing by Anni McHenry.

It is separate from Fieldlight Institute. Fieldlight Institute may cite, link, license, commission, purchase, or adapt work from this repository, but this repository preserves the authorship source of the writing itself.

## Migration Status

This repository is being built gradually from the public Say It Plain archive.

The migration process is intentionally one piece at a time:

1. Read the oldest available piece.
2. Decide where it belongs in the current structure.
3. Convert it into Markdown with source metadata.
4. Preserve the original Posthaven URL.
5. Commit the migrated piece with a clear authorship trace.

See [Migration Queue](MIGRATION_QUEUE.md).
